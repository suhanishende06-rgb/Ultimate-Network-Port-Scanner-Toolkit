
# 🌐 CyberScope - Web Based Network Port Scanner

CyberScope is a web-based Network Port Scanner built using Python and Flask.  
It allows users to scan open ports on a target IP address or domain through a simple and interactive web interface.

---

## 🚀 Features

- 🌐 Web-based interface
- ⚡ Multi-threaded fast scanning
- 🔍 Service detection
- 🎯 Scan custom port range
- 🛡️ Clean cyber-themed UI

---

## 🛠️ Technologies Used

- Python
- Flask
- Socket Programming
- HTML & CSS
- Multi-threading
